# VIM Using Notes


## First Level

- ` i `	// 按i进入Insert模式
- ` ESC `	// 按ESC返回Normal模式
- ` x `	// 删当前光标所在的一个字符
- ` :wq `	// 保存并退出(:w 保存, :q 退出)(w后面可以跟文件名)
- ` dd `	// 删除当前行(并把删除的行保存到剪贴板)
- ` p `	// 粘贴剪贴板
- ` hjkl `	// 移动光标，分别为『左』『下』『上』『右』(光标键` ← ↓ ↑ → `也可以)
- ` :help <command> `	// 显示相关命令的帮助(退出帮助需要` :q `)
